Как сделать button неактивным?

16.05.2020

Как деактивировать кнопку отправки формы?

В этой статье я покажу как просто с помощью checkbox сделать кнопку не активной. Деактивация кнопки пригодится например при отправке формы обратной связи и вам необходимо что бы пользователь поставил галочку что он прочитал "Пользовательское соглашение" и согласен с условиями.

HTML

Верстка простой формы: 

                
    <form>
        <input name="user-name" required="" placeholder="Ваше имя" type="text">
        <input name="user-email" required="" placeholder="Ваш E-mail" type="email">
        <input name="user-phone" required="" placeholder="Ваш E-mail" type="phone">
        <input id="checkbox" type="checkbox"><label>Принимаю пользовотальское соглашение</label>
        <button id="submit" type="submit" class="button button__form" disabled="">Отправить</button>
    </form>

CSS

Стили для формы: 

                
        form {
            position: absolute;
            top: 50%;
            left: 50%;
            transform: translate(-50%, -50%);
            display: flex;
            justify-content: center;
            align-items: center;
            flex-direction: column;
        }
        input {
            margin: 20px;
            padding: 10px;
        }
        
        button {
            margin: 20px;
        }

JavaScript

Скрипт: 

                
        window.addEventListener('DOMContentLoaded', function() {

            ' use strict ';

            let checkbox = document.getElementById('checkbox'),
                submit = document.getElementById('submit');
                    
                    submit.disabled=true; // При запуске кода сразу деактивируем кнопку
                    
                    checkbox.addEventListener('click', function() { // Событие клик
                        if (submit.disabled === true) { // Проверяем деактивированна кнопка или нет
                            submit.disabled=false; // Если была деактивированна, то активируем ее
                        } else {
                            submit.disabled=true; // И наоборот деактивируем если она была активна
                        }
                    });
        });

Таким способом кнопка будет не активна пока пользователь не установит галочку.
